## Run Locally

Clone the project

```bash
  git clone https://github.com/iamthakurnikhil/rick-and-morty.git
```

Go to the project directory

```bash
  cd rick-and-morty
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
