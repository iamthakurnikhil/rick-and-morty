## Run Locally

Clone the project

```bash
  git clone https://github.com/iamthakurnikhil/movie-website.git
```

Go to the project directory

```bash
  cd movie-website
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
